Sales Dashboard – Power BI / Tableau

This project presents a simple and interactive Sales Dashboard built using Power BI/Tableau.
It visualizes sales performance across month, region, and product category using the Superstore dataset.

 Project Contents

Dashboard.jpg – Screenshot of the final sales dashboard

sales_dashboard_insights.txt – Key insights extracted from the dashboard

Power BI / Tableau file (optional, if you include it)

 Objective

To create a basic but effective dashboard displaying:

Monthly sales trends

Sales by region

Sales by product category

A filter/slicer for better interactivity

 Tools Used

Power BI or Tableau

(Optional) Python + Pandas for data cleaning

 Dashboard Features

Line Chart: Sales over time (Month–Year)

Bar Chart: Sales by region

Donut Chart: Sales by category

Slicer: Region or category filter for interactive analysis

 Key Insights

The South region recorded the highest total sales, while the West had the lowest.

Technology contributed the largest share of category-wise sales.

Monthly sales fluctuate significantly, with clear highs in certain months.

All three product categories contribute almost equally to the overall revenue.
